## 🏗️ Featured Projects

### 🤖 [Agentic Document Understanding Pipeline](#)
> **"NER, Relation Extraction, and self-correction in a multi-step agentic loop."**

A LangChain-based pipeline that extracts named entities and structured relations from unstructured text documents.

- **Framework:** LangChain with tool-use and self-correction loops.
- **Tasks:** Named Entity Recognition, Coreference Resolution, Summarization.
- **Evaluation:** LLM-as-a-Judge methodology for hallucination and drift detection.

### ⚡ [ML Inference Optimization Pipeline](#)
> **"Pruning, QLoRA, and quantization benchmarked across NLP tasks."**

Built and benchmarked CNN and transformer models with inference optimization techniques in PyTorch.

- **Techniques:** LoRA fine-tuning, QLoRA, quantization, few-shot & zero-shot evaluation.
- **Tools:** Hugging Face Transformers, PyTorch, structured benchmarking reports.
- **Evaluation:** LLM-as-a-Judge for output quality assessment.

### 🔧 [DSP Time-Shift / Time-Reversal Module](#)
> **"Real-time streaming signal processing in SystemVerilog."**

Implemented a real-time DSP memory module supporting time-shift and time-reversal operations.

- **Hardware:** SystemVerilog, low-latency buffering and control logic.
- **Verification:** Simulation testbenches covering edge cases and timing constraints.

### 🖥️ [5-Stage Pipelined MIPS Processor & Cache Simulator](#)
> **"Cycle-accurate CPU simulation with full hazard detection."**

Built a MIPS pipeline simulator in C++ with data forwarding, hazard detection, and a 2-bit branch predictor.

- **Memory:** Configurable cache simulator with LRU/FIFO/Random replacement policies.
- **Validation:** Systematic trace-based testing across all edge cases.
